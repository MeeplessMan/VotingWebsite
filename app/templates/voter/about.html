About This Application

This application is designed to replace the current DUT SRC Election system. It aims to provide a more efficient and user-friendly platform for conducting elections at DUT.

The election system supports two campuses: Midlands and Durban. Voters from the Durban campus can select up to 8 candidates, while voters from the Midlands campus can select up to 7 candidates.
